## Task 3: Exploratory Data Analysis (EDA) 


---

### **Files in this Repository**

#### **1. Analysis Notebook**

* **`task3.ipynb`**: The main Jupyter Notebook containing the Python code for data cleaning, processing, and generating the visualizations listed below.

#### **2. Visualizations (Images)**

The following plots provide insights into the dataset:

* **Content Distribution**
* `Distribution_of_movies_by_type.png`: A breakdown of Movies vs. TV Shows.
* `Top_10_types_most_popular.png`: Insights into the most frequent genres or categories.


* **Timeline Trends**
* `distribution_of_movies_by_release_year.png`: Shows how content volume has changed over the years.
* `Distribution_of_release_years_by_count.png`: A statistical view of content density across different years.
* `Distribution_of_release_years_by_type.png`: A comparison of release trends between Movies and TV Shows.


* **Ratings & Geography**
* `Distribution_of_Movies_by_Rating.png`: Analysis of audience maturity ratings (e.g., PG-13, TV-MA).
* `Top_10_Countries_with_the_Most_Movies.png`: Identifies the leading countries in content production.


* **Heatmaps (Correlations)**
* `Type_vs_country_heatmap.png`: Visualizes the relationship between content type and the country of origin.
* `Type_vs_rating_heatmap.png`: Visualizes how ratings are distributed across Movies and TV Shows.


####  **3. Datasets**
* This folder contains the dataset named: `netflix_titles.csv`
