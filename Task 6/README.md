# House Price Prediction using Linear Regression

## Description  
This project uses **Linear Regression** to predict house prices using the **California Housing dataset**.  
The notebook loads the dataset, trains a model, and evaluates its performance.

---

## Steps Performed

1. Import required libraries (NumPy, Pandas, Matplotlib, Scikit-learn).  
2. Load the California Housing dataset and convert it to a DataFrame.  
3. Display basic data information using:
   - `head()`
   - `info()`
   - `describe()`  
4. Separate:
   - Features (X)  
   - Target variable (y = Price)  
5. Split the dataset into training and testing sets.  
6. Train a Linear Regression model.  
7. Predict house prices on test data.  
8. Compare actual vs predicted values.  
9. Evaluate the model using:
   - MAE (Mean Absolute Error)  
   - RMSE (Root Mean Square Error)  
10. Plot actual vs predicted values using a scatter plot.  
11. Show model coefficients to find important features.

---

## Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## Output

- Predicted house prices  
- Error metrics (MAE, RMSE)  
- Scatter plot of Actual vs Predicted values  
- Feature importance using model coefficients
